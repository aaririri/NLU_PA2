# B22AI001 – Programming Assignment 2

## Overview

This repository contains solutions for two NLP tasks:

- **Problem 1:** Word Embeddings (CBOW, Skip-gram) using IIT Jodhpur corpus  
- **Problem 2:** Character-Level Name Generation using RNN variants  

---

## Files


prob1.py Problem 1 implementation (uses corpus.txt)
prob2.py Problem 2 implementation (uses TrainingNames.txt)
corpus.txt Text corpus for Word2Vec training
TrainingNames.txt Dataset of Indian names


---

## Usage

### Problem 1: Word Embeddings
```bash
python prob1.py
```
### Problem 2: Name Generation
```bash
python prob2.py
```
### Requirements
```bash
pip install numpy matplotlib scikit-learn gensim wordcloud pymupdf torch
```
